
- どんなことができるアプリか
四則演算ができる電卓のアプリです。
ユーザーが「n」を入力するまで処理を継続できるよう、while文によるループ処理を実装しています。
0で除算しようとした場合や、数値・演算子以外の入力が行われた場合には例外処理を行い、
エラーが発生してもアプリが終了せず継続できるようにしています。

- 実行方法 
python calculator.py

- 使用したPythonのバージョン  
3.13.2

- 工夫したポイント
第三者が見ても分かりやすいよう、インデントを揃え、変数名も処理内容が分かるものを意識して設定しました。
finally句を使用することで、answer = input("計算を続けますか？(y/n)") の処理が重複しないように工夫しました。  
想定していない例外が発生した場合でも処理が停止しないよう、except Exception を用いて「予期せぬ例外が発生しました」と表示する処理を実装しました。

- 苦労した点や未実装の機能（あれば）
四則演算の記号が文字列として扱われることを理解するまでに時間がかかりました。
無効な演算子が入力された場合の例外処理の実装方法に苦労しました。
過去の計算履歴の表示機能とテストコードが未実装です。
